Simple Web Chat

Goal was a basic web chat app where users could send simple text based messages to each other. The main objectives were:

  • Create a Rest Node Server with http routes
  • Connect this to a MongoDB server instance for data persistence
  • Create a simple front end with React components for displaying data in message like format and user interaction
  • Implement Rest API to connect with the server using the http routes

Tools Used:

  • HTML
  • Javascript
  • Node.js
  • React
  • MongoDB
  • Express
