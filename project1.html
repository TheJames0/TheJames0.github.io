Vulkan Physics Engine Project

The goal of the project is to create a physics engine that runs largely on the GPU using Vulkan. The main objectives were:

  • Physical Accuracy using higher accuracy numerical integration such as RK4 integration
  • Easily Accessible and high levels of interactivity with manipulation tools and easy settings
  • Handle a variety of shapes created in CAD or other 3D modelling tools
  • Fast and scalable to simulate hundreds of detailed objects and constraints using GPU compute shaders by only accessing information when required by the GPU

Tools Used:

  • C++
  • Vulkan
  • GLSL
  • GLM
  • ImGui
