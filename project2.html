Marketplace application

The goal of this project was to make a demo project of a marketplace like gumtree or ebay. The main objectives were:

  • User account creation
  • User posting linked with accounts
  • Viewing posts
  • Editing with Authentication
  • Image upload and storage

Tools used

  • C#
  • ASP.NET
  • Azure Identity Framework
  • Azure Cloud Services
