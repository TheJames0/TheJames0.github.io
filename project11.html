Mobile Hex Strategy

An alien-themed hex-based strategy project with procedural map generation, turn-based combat, resource and building systems, and multiplayer support across mobile and desktop.

  • Procedural hex tile map generation.
  • Turn-based unit selection, movement, and actions with multiplayer synchronization.
  • Support for both mobile and desktop platforms.
  • Resource and building systems for progression.
  • Custom level editor integrated into Unity.
  • Complex movement and action validation for each unit type.
